# AI Smart Resume & Cover Letter Tailor

## Overview
AI Smart Resume & Cover Letter Tailor is a Python-based web application that helps users generate ATS-friendly resumes and professional cover letters based on a job description.

## Features
- Upload Resume in PDF format
- Extract text using PDFPlumber
- Generate ATS-friendly resume
- Generate professional cover letter
- Interactive Streamlit web interface

## Technologies Used
- Python
- Streamlit
- OpenAI API
- PDFPlumber

## How to Run
1. Clone the repository.
2. Install the required libraries:
   pip install -r requirements.txt
3. Add your OpenAI API key in Streamlit Secrets.
4. Run the application:
   streamlit run App.py
